# Theme System POC

A proof of concept for a CSS token-based theming system with a live inspector.

## What it demonstrates

- **CSS custom properties** as design tokens (colors, typography)
- **Per-theme token overrides** via `[data-theme]` attribute
- **Tailwind Variants** for component variant management
- **Live inspector** — switch themes and watch tokens update in real time

## Stack

- Vue 3 + TypeScript
- Tailwind CSS v4
- Tailwind Variants
- Reka UI primitives

## Run it

```sh
pnpm install
pnpm dev
```
